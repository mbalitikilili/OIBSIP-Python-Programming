# BMI Calculator

## Description
This is a simple Python program that calculates a user's Body Mass Index (BMI) based on their weight and height.

## Features
- Accepts weight in kilograms (kg)
- Accepts height in meters (m)
- Calculates BMI
- Displays the BMI category:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Handles invalid input using error checking

## Technologies Used
- Python 3

## How to Run
1. Open the project in VS Code.
2. Run:
   ```
   python bmi_calculator.py
   ```
3. Enter your weight and height when prompted.
