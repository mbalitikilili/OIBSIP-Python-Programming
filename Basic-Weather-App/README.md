# Basic Weather App

## Description
A Python application that retrieves real-time weather information using the OpenWeather API.

## Features
- Search weather by city
- Displays:
  - Temperature
  - Weather condition
  - Humidity
- Handles invalid city names
- Handles network errors
- Allows multiple searches

## Technologies Used
- Python 3
- Requests Library
- OpenWeather API

## How to Run
1. Install the requests library:
   ```
   pip install requests
   ```
2. Add your OpenWeather API key to the program.
3. Run:
   ```
   python weather_app.py
   ```
